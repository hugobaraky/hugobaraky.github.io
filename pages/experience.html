Understanding Integrated Gradients

Integrated Gradients (IG) is a method for attributing the prediction of a deep network to its input features. It provides a way to understand which features are most influential in the model's decision-making process.

Mathematical Definition

The Integrated Gradients for the ith input feature is defined as:

$$ IG_i(x) = (x_i - x'_i) \times \int_{\alpha=0}^{1} \frac{\partial F(x' + \alpha \times (x - x'))}{\partial x_i} d\alpha $$

Where:

  • x is the input vector.
  • x' is the baseline input (e.g., all zeros).
  • F is the model's output function.
  • Î± is a scalar that interpolates between the baseline and the input.

Discrete Approximation

In practice, the integral is approximated using a summation:

$$ IG_i(x) \approx (x_i - x'_i) \times \frac{1}{m} \sum_{k=1}^{m} \frac{\partial F(x'_k)}{\partial x_i} $$

Where m is the number of steps in the approximation.
